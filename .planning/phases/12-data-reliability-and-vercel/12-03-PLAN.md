# Phase 12: Data Reliability & Vercel Compatibility
## Plan 12-03: Frontend Market & Macro Data Displays

### Objective
Update the frontend Dashboard and Report views to visually display the new Market (Yahoo Finance/Alpha Vantage) and Macro (TCMB/TÜİK) data that the pipeline now gathers.

### Context & Rationale
We added the Market and Macro agents in Phase 10, and they now successfully save data into `agent_runs` (Phase 11). However, the frontend UI currently only shows the final Analyst summary and raw KAP/News tables. We need dedicated UI components to display the stock price, PE ratio, USD/TRY exchange rate, and inflation data to give the user a complete dashboard experience.

### Scope
- Create a `MarketDataCard` component to display the stock's current price, daily change, and key statistics (PE, Market Cap).
- Create a `MacroDataCard` component to display the latest USD/TRY, Inflation (TÜFE), and Interest Rate.
- Integrate these components into the `ReportView` (e.g., `src/app/dashboard/reports/[id]/page.tsx`).

### Implementation Steps
1. **Component Creation**: Build the React components in `src/components/dashboard/` using Tailwind CSS and existing UI libraries.
2. **Data Fetching**: Update the report page to fetch the `agent_runs` for the specific `session_id` and extract the Market and Macro data.
3. **Integration**: Render the cards prominently at the top of the report view, above the AI Analyst synthesis.

### Verification (Goal-Backward)
