### Task 1: Implement KAP Retry with Exponential Backoff
**File:** `src/lib/kap/client.ts`
**Changes:** Add a `fetchWithRetry` helper and apply it to both KAP methods.

```typescript
async function fetchWithRetry(
  url: string,
  options: RequestInit,
  maxRetries: number = 2,
  baseDelayMs: number = 2000
): Promise<Response> {
  let lastError: Error | null = null;
  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    try {
      const response = await fetch(url, {
        ...options,
        signal: AbortSignal.timeout(10000), // 10s per attempt (was 15s)
      });
      return response;
    } catch (error) {
      lastError = error as Error;
      if (attempt < maxRetries) {
        const delay = baseDelayMs * Math.pow(2, attempt);
        console.warn(`[KAP Client] Attempt ${attempt + 1} failed, retrying in ${delay}ms...`);
        await new Promise(r => setTimeout(r, delay));
      }
    }
  }
  throw lastError;
}
```

**Apply to:**
- `fetchViaDisclosureQuery` (line 95-106): Replace `fetch(...)` with `fetchWithRetry(...)`
- `fetchViaRecentDisclosures` (line 132-139): Replace `fetch(...)` with `fetchWithRetry(...)`
- Remove the `signal: AbortSignal.timeout(15000)` from fetch options (now handled inside `fetchWithRetry`)

**Verification:** KAP calls retry on timeout; logs show `Attempt 1 failed, retrying in 2000ms...`

### Task 2: Relax News Agent Relevance Filter
**File:** `src/agents/news-agent.ts`
**Changes at line 97-109:**

Replace the strict filter with a soft fallback strategy:

```typescript
// 2.5 Relevance Filter — strict for tickers, but with fallback
let filteredRawData = uniqueRawData;
if (queryType === 'ticker') {
  const strictFiltered = uniqueRawData.filter(article => {
    const content = `${article.title || ''} ${article.description || ''}`.toLowerCase();
    const tickerLower = ticker.toLowerCase();
    const companyName = getCompanyName(ticker);
    const hasTicker = content.includes(tickerLower);
    const hasCompanyName = companyName ? content.includes(companyName.toLowerCase()) : false;
    return hasTicker || hasCompanyName;
  });

  if (strictFiltered.length > 0) {
    filteredRawData = strictFiltered;
  } else {
    // Fallback: if strict filter returns 0, take top 5 unfiltered as general context
    console.log(`[News Agent] No exact matches for "${ticker}", using top ${Math.min(5, uniqueRawData.length)} general articles as fallback.`);
    filteredRawData = uniqueRawData.slice(0, 5);
  }
}
```

**Verification:** When exact ticker match yields 0 results, news agent returns up to 5 general Turkish financial articles instead of empty array.
