### Task 1: Formalize Market Agent Provider Chain with Provenance
**File:** `src/agents/market-agent.ts`
**Changes:**

Restructure the provider chain with explicit logging and provenance:

```typescript
export async function runMarketAgent(...): Promise<MarketData | null> {
  // Tier 1: RapidAPI Yahoo providers (3 sub-providers)
  try {
    const rtData = await fetchYahooMarketData(ticker);
    if (rtData) {
      console.log(`[Market Agent] Tier 1 success: ${rtData.source?.provider}`);
      // ... save and return
    }
  } catch { /* continue to tier 2 */ }

  // Tier 2: Yahoo Finance v3 (direct)
  try {
    const yahooData = await getYahooMarketData(ticker);
    if (yahooData) {
      console.log(`[Market Agent] Tier 2 success: yahoo_finance_v3`);
      // ... save and return
    }
  } catch { /* continue to tier 3 */ }

  // Tier 3: Alpha Vantage
  try {
    const avQuote = await getStockQuote(ticker);
    const avOverview = await getCompanyOverview(ticker);
    // ... combine and return with source: 'alpha_vantage'
  } catch { /* all tiers failed */ }

  return null;
}
```

Key changes:
- Import `getYahooMarketData` from `yahoo-client.ts` (Tier 2) separately from `getFullMarketData`
- Import `getStockQuote` and `getCompanyOverview` directly for Tier 3 (skip `getFullMarketData` wrapper to avoid double Yahoo call)
- Each tier tags `source.provider` explicitly
- Remove the current `getFullMarketData` call which double-triggers Yahoo → Alpha Vantage internally

**Verification:** Logs show attempted tiers in order; successful tier identified by provider name.
