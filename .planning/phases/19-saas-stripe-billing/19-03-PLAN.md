<objective>
Add an "Abonelik" (Subscription) tab to the dashboard Settings page.
Users can see their current plan status and navigate to Stripe Customer Portal to manage it.
</objective>

<context>
## Current Settings Page
- `src/app/dashboard/settings/page.tsx` — server component, fetches `user_settings` from Supabase
- `src/components/dashboard/SettingsForm.tsx` — client component with theme + notifications sections
- No tab UI exists yet — the whole page is just the SettingsForm

## What the Abonelik Tab Must Show
1. **Current Plan Badge** — e.g. "PRO" badge with green glow
2. **Token Balance** — "87 / 100 araştırma kullanıldı" with progress bar
3. **Renewal Date** — "Sonraki yenileme: 19 Haziran 2026" (from `current_period_end`)
4. **Cancel Warning** — If `cancel_at_period_end = true`: "Aboneliğiniz dönem sonunda iptal edilecek." warning box
5. **Action Buttons**:
   - Free tier: "Planı Yükselt →" → links to `/#pricing`
   - Paid tier (active): "Aboneliği Yönet" → POST to `/api/stripe/portal` (Stripe Customer Portal)
   - Past due: "Ödemeyi Güncelle" → same portal link

## Tab UI Pattern
Use a simple client-side tab switcher (no router changes):
- Two tabs: "⚙ Genel Ayarlar" and "💳 Abonelik"
- Active tab has green underline border
- Tab state managed with `useState`
- This keeps the URL clean and avoids unnecessary routing complexity

## Data Fetching
The settings page (server component) should fetch BOTH `user_settings` AND the user's subscription data from `users` table:
- `tier`, `subscription_status`, `current_period_end`, `cancel_at_period_end`, `tokens_balance`
Pass both datasets as props to a new client wrapper component.
</context>

          activeTab === 'general'
            ? 'text-[#22c55e] border-b-2 border-[#22c55e]'

      >

  <action>
    Update the server component to also fetch the user's subscription data from the `users` table:

    After the existing `user_settings` fetch, add:
    ```ts
    let subscriptionData = {
      tier: 'free' as Tier,
      subscriptionStatus: 'inactive' as SubscriptionStatus,
      tokensBalance: 5,
      currentPeriodEnd: null as string | null,
      cancelAtPeriodEnd: false,
    };

    if (userId) {
      const { data: userProfile } = await supabase
        .from('users')
        .select('tier, subscription_status, tokens_balance, current_period_end, cancel_at_period_end')
        .eq('id', userId)
        .single();

      if (userProfile) {
        subscriptionData = {
          tier: userProfile.tier ?? 'free',
          subscriptionStatus: userProfile.subscription_status ?? 'inactive',
          tokensBalance: userProfile.tokens_balance ?? 5,
          currentPeriodEnd: userProfile.current_period_end ?? null,
          cancelAtPeriodEnd: userProfile.cancel_at_period_end ?? false,
        };
      }
    }
    ```

    Pass it to `SettingsForm`:
    ```tsx
    <SettingsForm initialSettings={initialSettings} subscription={subscriptionData} />
    ```
