<objective>
Replace Alpha Vantage as the primary BIST market data source with Yahoo Finance, which has substantially better BIST.IS coverage and no daily quota constraint. Add a short-TTL in-memory cache so the dashboard does not re-hammer providers when the same ticker is re-requested.
</objective>

<context>
Diagnostic finding from hotfix testing: `npx tsx src/agents/market-agent.ts THYAO` produces `Cannot read properties of undefined (reading '01. symbol')` because Alpha Vantage's GLOBAL_QUOTE returns an empty object for BIST tickers under the free tier. Yahoo Finance's `yahoo-finance2` npm package handles BIST tickers (suffix `.IS`) reliably and is free.
</context>
