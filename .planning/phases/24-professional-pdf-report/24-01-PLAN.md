# Plan 22-01 — Kütüphane Migrasyonu: jsPDF → @react-pdf/renderer + Türkçe Font Desteği

## Goal
Mevcut jsPDF tabanlı PDF üretecini tamamen `@react-pdf/renderer` ile değiştir. Bu kütüphane:
- Yerleşik Unicode / UTF-8 desteği ile Türkçe karakterleri (ş, ı, ğ, ü, ö, ç, İ, Ş, Ğ) bozulmadan render eder
- React bileşen modeli ile karmaşık sayfalı düzenleri kolayca yönetir
- Vercel serverless ortamında çalışır (Puppeteer gerektirmez)

## Context
**Sorun (Root Cause):**
- `src/lib/pdf/generator.ts` jsPDF + helvetica kulllanıyor; bu font Latin-1 tabanlı, Türkçe extended karakterleri desteklemiyor
- AI ürettiği Markdown (`#`, `*`, `**`) ham string olarak `addBodyText()` fonksiyonuna geçiyor, render edilmiyor
- `ReportData` tipinde `investmentRecommendation` alanı yok; analist bunu üretiyor ama PDF'e yansımıyor
- Karakter başına boşluk sorunu (`A S E L S A N`): jsPDF'in font metrics problemi

**Hedef:** 10–12 sayfalık, kurumsal görünümlü, Türkçe karakterlerin tamamen doğru çıktığı PDF.

### Task 4: Yeni PDF Generator Yaz (react-pdf)
**File:** `src/lib/pdf/generator.ts` (komple yeniden yaz)

Mevcut dosyayı sil, yeni implementasyonu yaz:

```typescript
import React from 'react';
import { renderToBuffer, Document, Page, View, Text, StyleSheet, Font } from '@react-pdf/renderer';
import { ReportData } from '../../types/research';
import * as fs from 'fs';
import * as path from 'path';

// NotoSans fontunu proje içine embed et (Google Fonts'tan indir)
Font.register({
  family: 'NotoSans',
  fonts: [
    { src: path.join(process.cwd(), 'public/fonts/NotoSans-Regular.ttf') },
    { src: path.join(process.cwd(), 'public/fonts/NotoSans-Bold.ttf'), fontWeight: 'bold' },
  ]
});

export async function generatePdfReport(data: ReportData, outputPath: string): Promise<void> {
  const buffer = await renderToBuffer(<ReportDocument data={data} />);
  const dir = path.dirname(outputPath);
  if (!fs.existsSync(dir)) fs.mkdirSync(dir, { recursive: true });
  fs.writeFileSync(outputPath, buffer);
}
```

### Task 5: Font Dosyalarını Ekle
`public/fonts/` dizinine şu iki dosyayı indir ve yerleştir:
- `NotoSans-Regular.ttf` — Google Fonts Noto Sans v36 Regular
- `NotoSans-Bold.ttf` — Google Fonts Noto Sans v36 Bold

İndirme komutu (build sırasında otomatik):
```bash
mkdir -p public/fonts
curl -L "https://github.com/google/fonts/raw/main/ofl/notosans/NotoSans%5Bwdth%2Cwght%5D.ttf" -o public/fonts/NotoSans-Regular.ttf
```

Alternatif: `next.config.ts`'de download script'i tanımla; ya da statik dosyaları proje reposuna commit et.
